# Task Tracker (TypeScript CLI)

## Overview
This is a simple **command-line task tracker** built using **TypeScript** and **Node.js**.  
It allows users to manage tasks directly from the terminal.

All tasks are stored **in memory** using an array, so data is available only while the program is running.



## Features
- Add a task
- List all tasks
- Mark a task as completed
- View completed tasks
- View pending tasks
- Exit the application



## Technologies Used
- TypeScript
- Node.js
- readline module

## How to Run
```bash
npm install
npx tsc
node dist/index.js
